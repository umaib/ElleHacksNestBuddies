--Program: Nest Buddies web application
--Original Authors: this project was created for ElleHacks 2019 alongside a group of three other students (Emma Tsai, Ashley Chen, Amrita Maharaj) and myself (Umai Balendra).

--Purpose of Program: to provide the user with the option to either host a room available in their home, or search for a room to rent out. The intended audience is seniors to aid the effort in reducing homelessness amongst the senior population.
